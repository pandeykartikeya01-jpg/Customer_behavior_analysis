Customer Behavior Analysis

This project analyzes customer shopping behavior using Python, SQL, and Power BI. The workflow includes loading the dataset in Python, performing exploratory data analysis (EDA), cleaning the data, running analytical SQL queries, and building an interactive Power BI dashboard to generate business insights.

The dataset contains customer shopping data including:
1. Customer demographics (age, gender)
2. Product categories and items purchased
3. Purchase amount
4. Review ratings
5. Discount usage
6. Subscription status
7. Shipping method
8. Previous purchases
The dataset is used to analyze purchasing patterns, revenue contribution, and customer segments.

Tools Used:
1. Python (Pandas) – Data loading, cleaning, and EDA
2. SQL (MySQL) – Analytical queries and customer segmentation
3. Power BI – Dashboard creation and visualization
4. Jupyter Notebook – Data analysis environment

Steps:
1. Data Loading:
The dataset was loaded into Python using Pandas.

2. Exploratory Data Analysis (EDA):
Initial exploration was performed to understand:
Data structure
Missing values
Data distributions
Customer behavior patterns

3. Data Cleaning:
Cleaning steps included:
Handling missing values
Formatting data types
Preparing the dataset for analysis

4. SQL Analysis:
The cleaned dataset was stored in MySQL and SQL queries were used to answer business questions such as:
Revenue by gender
Product performance
Customer segmentation
Discount usage patterns

5. Dashboard Creation:
An interactive Power BI dashboard was created to visualize key insights.

Key insights from the analysis include:
Clothing category generated the highest revenue.
Young adult customers contribute the largest share of sales.
A significant percentage of customers are non-subscribers.
Discount usage influences purchasing behavior.
